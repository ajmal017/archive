# website-gmc

A Vue.js website for Global Media Campaign. It also has some firebase cloud functions to the mailchimp and youtube api functionality.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
